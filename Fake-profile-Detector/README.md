# Fake Profile Detector

This project detects fake social media profiles using OSINT-based techniques.

## Features
- Profile URL analysis
- Profile image verification
- Fake / Genuine classification
- Automatic PDF report generation

## Technologies Used
- Python
- BeautifulSoup
- Requests
- HTML, CSS, JavaScript

## How to Run
1. Navigate to backend folder
2. Install requirements:
   pip install -r requirements.txt
3. Run:
   python main.py

## Output
- Console verdict
- PDF report generated in Report folder
